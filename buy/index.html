Frequently asked questions

What does "department license" mean?
One license covers everyone in a single academic department who shares the same TermPoint database. There's no limit on the number of users — if someone new joins, they're covered automatically.
Do I need a separate license for each database?
The license file needs to be present in each folder that contains a TermPoint database. If your department uses one database, one copy of the license file is all you need. If you maintain separate databases (e.g. one per subject area), place a copy of the same license file in each folder.
What happens when the license expires?
TermPoint switches to read-only mode — you can still open and view your schedule, but editing is disabled until you renew. Your data is never locked away.
Is an internet connection required?
No. The license is verified locally using a digital signature — no activation server, no phone-home, no internet requirement. TermPoint works the same way on an air-gapped network.
What if I want to try before I buy?
TermPoint includes a 30-day free trial with full editing access. Just install and start scheduling — no credit card, no sign-up. Purchase a license when you're ready.
Can I get a quote or invoice for my institution's procurement process?
Yes — contact us at admin@termpoint.ca and we'll provide whatever your purchasing office needs.
